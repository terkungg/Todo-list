ตั้งค่าการเชื่อมต่อระบบคลาวด์ (Cloud Sync)

ระบบนี้เชื่อมต่อกับ **Firebase Firestore** เพื่อบันทึกข้อมูลงานออนไลน์ของคุณ ทำให้คุณสามารถเปิดดูและแก้ไขงานจากเครื่องไหนก็ได้โดยไม่ต้องตั้งค่าใหม่

สถานะปัจจุบัน: กำลังใช้งานฐานข้อมูลร่วมกัน
วิธีการตั้งค่าฐานข้อมูลของคุณเอง (ใช้งานฟรี 100%)
  1. เข้าไปที่ Firebase Console ด้วยบัญชี Google
  2. กด **Create a project** แล้วตั้งชื่อโปรเจกต์ (เช่น MyTodoApp)
  3. เลือกเมนู **Web ()** เพื่อสร้างเว็บแอปพลิเคชัน คัดลอกข้อมูลส่วน `firebaseConfig`
  4. ในเมนูด้านซ้าย ไปที่ **Firestore Database** แล้วกด **Create database**
  5. เลือกเริ่มในโหมดทดสอบ (**Start in test mode**) เพื่อให้เขียนอ่านข้อมูลได้ฟรีทันที
  6. นำค่า Config JSON หรือค่าคีย์ต่างๆ มาป้อนในช่องด้านล่างนี้
